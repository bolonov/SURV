Общее описание системы учета рабочего времени

В левой части экрана системы располагается основной раздел "Кто в офисе" - это список сотрудников находящихся в текущий момент в офисе

     

Наверху формы доступны:

  • дополнительные фильтры
    • по офису (по умолчанию устанавливается отбор по офису текущего пользователя)
    • по сотруднику

Работает поиск по первым буквам фамилии сотрудника и названия офиса:

  • меню отчетов (см. далее)

 

В правом верхнем углу расположен список заявок пользователя, размещенных в системе для одобрения ответственным сотрудником:

      

Функции кнопок над формой:

  • Создать - создает новую заявку (см. далее)
  • Удалить - удаляет завку, на которой стоит курсор
  • Показать все - показывает в т.ч. ранее рассмотренные заявки (увержденные/отклоненные)
  • Обновить - обновляет список

Функции кнопок над формой:

  • Утвердить - утверждает заявку, на которой стоит курсор
  • Отклонить - отклоняет завку, на которой стоит курсор
  • Показать все - показывает в т.ч. раннее рассмотренные заявки (увержденные/отклоненные)
  • Обновить - обновляет список

Работа с заявками на отсутствие

При размещении заявки в форме заявки необходимо отразить:

  • Вид отсутствия
  • Период начала и окончания (для некоторых видов отсутствия система даст ввести даты с/по, а для некоторых только одну дату и время с/по)
  • Пользователя, согласующего заявки (в зависимости от Вида отсутствия это могут быть непосредственные руководители, кадровики или сам пользователь если ему предоставлено соответствующее право)
  • Произвольный комментарий

После размещения заявка появится в списке заявок ответственного пользователя, а после  отражаться утверждения/отклонения могут быть показаны с помощью нажатия кнопки "Показать все".

При размещении, одобрении или отклонении заявки на эл. почту ответственного и автора заявки приходит соответствующее уведомление.

Отчеты

В меню отчеты доступны 2 отчета по регистру учета времени и 1 отчет по заявкам об отсутствии (вывод данных доступен по самому пользователю и по его подчиненным):

     

  • "Отработанное время" показывает данные по отработанному и стандартному производственному календарю. С помощью различных вариантов этого отчета (используется кнопка Выбрать вариант) можно определить количество часов недо/переработки за выбранный период с различной детализацией:

     

  • "Ошибки учета" показывает ошибки учета времени при регистрации по карточкам (повторные вход/выход, отсутствие выхода):

     

  • "Заявки об отсутствии" показывает все доступные заявки с различными фильтрами
